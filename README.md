# Snake Game

This is a bite-sized remake of the iconic Snake arcade game built using plain HTML, CSS, and JavaScript. It runs entirely in the browser with no build tooling or dependencies required.

## Features

- Smooth 20x20 grid gameplay rendered on a `<canvas>`
- Keyboard support for arrows/WASD, space to pause, and `R` to restart
- Simple touch controls for swipes on mobile screens
- Responsive layout with a modern neon aesthetic

## Getting Started

1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd snake-game
   ```
2. Open `index.html` in your favorite browser (double-click, or serve statically with `npx serve .`).
3. Start playing immediately.

## Controls

- **Move**: Arrow keys or W/A/S/D
- **Pause / Resume**: Space bar or the on-screen button
- **Restart**: Click the restart button or press `R` after a game over
